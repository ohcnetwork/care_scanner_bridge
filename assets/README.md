# Icon Assets

Place your icons in this directory:

## Required Icons

### System Tray Icon
- **Filename**: `icon.png`
- **Size**: 22x22 or 32x32 pixels (PNG, with transparency)
- **Usage**: System tray icon on all platforms

### macOS App Icon
- **Location**: `build/darwin/icon.icns`
- **Format**: Apple ICNS format
- **Sizes**: 16, 32, 64, 128, 256, 512, 1024 (all in one .icns file)

### Linux App Icon
- **Location**: `build/linux/icon.png`
- **Size**: 256x256 pixels (PNG)

## Creating Icons

You can use tools like:
- **macOS**: `iconutil` to convert PNG to ICNS
- **Online**: https://cloudconvert.com/png-to-icns
- **Icon generator**: https://www.iconfinder.com/

## Suggested Design

A simple barcode scanner icon works well. Consider using:
- A barcode with scan lines
- A scanner device silhouette
- The Care logo integrated with a scanner symbol
